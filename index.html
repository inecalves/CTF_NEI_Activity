Workshop Cybersecurity: CTF Edition

Missão: Encontra as 5 flags escondidas no sistema.

Nível Fácil

1. Existe uma flag comentada no código desta página.

2. Existe uma flag escondida num ficheiro externo (.css) que o site carrega.

Nível Médio

3. HTTP Header: Inspeciona o tráfego de rede (Network) para encontrar a flag enviada pelo "servidor".

4. Cookies: Altera o valor de 'admin' para 'true' para libertar a flag.

SUCESSO: FLAG_4 {COOKIE_MANIPULATION_PRO}

Nível Difícil

5. SQL Injection: Faz bypass ao login para entrar como administrador.
